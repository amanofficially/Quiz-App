# 🧠 Quiz App — React + Vite + Tailwind CSS

A fully functional Quiz Application built with React, Vite, and Tailwind CSS during my MERN Stack Internship at SVVV, Indore.

## ✨ Features
- 8 MCQ questions about React, Vite & Tailwind CSS
- Progress bar showing completion
- Green highlight for correct, red for wrong answer
- Live score counter during quiz
- Result screen with score and percentage
- Try Again button to restart

## 🛠️ Tech Stack
- React.js
- Vite
- Tailwind CSS
- JavaScript ES6+

## 🧠 React Concepts Used
- useState — screen navigation, score, selected answer
- Conditional rendering — start, quiz, result screens
- .map() — rendering answer buttons
- Conditional Tailwind classes — green/red button colors
- Props — passing questions and callbacks
